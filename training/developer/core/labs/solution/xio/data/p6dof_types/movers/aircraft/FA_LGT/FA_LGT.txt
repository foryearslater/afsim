# FA-LGT (Fighter/Attack-Light) :  This is probably one of the most common FA aircraft for unclassified scenarios. 
# The FA-LGT is roughly similar in size and performance to an F-16 or a MiG-29 (but with a single engine). 
# It uses a conventional wing-tail configuration. This model is closer to the F-16 in size/dimensions since it only has a single engine. 
# All FA-LGT series models will have an empty weight of 20,000 lbs and an internal fuel weight of 5,000 lbs.
# 
# Length:    54 ft (F-16: 50 ft vs MiG: 57 ft)
# Wingspan:  35 ft (F-16: 33 ft vs MiG: 37 ft)
# Empty Wgt: 20,000 lbs (F-16: 18,000 lbs vs MiG: 24,000 lbs)
# Fuel Wgt:  5,000 lbs (internal)
# Wing Area: 330 sq-ft (F-16: 300 sq-ft vs MiG: 400 sq-ft)
#   
# Engine: 1x TurbofanFighter Engine (16,000 MIL and 26,000 lbs A/B)
# This compares to the F-16 (15,000-17,000 MIL and 24,000-28,000 lbs A/B) and the MiG-29 2x(11,000 MIL and 18,000 lbs A/B each).
#                       
# ================================================

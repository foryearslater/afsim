.. This file is provided to serve as a generation target, allowing Warlock demos to be linked into
.. other documentation when they are available to the build system, without creating a build dependency.
.. To build the documentation with demos, enable WSF_INSTALL_DEMOS in CMake.
.. To include a list of Warlock related demos into a document, use the following markup: 
.. .. include::`warlock_demos.txt`
.. Any file named _warlock_demos.txt will be appended to this file during generation.
.. Files named _warlock_demos.txt should only be present in the demos repository.
.. Content should not be added directly to this file.

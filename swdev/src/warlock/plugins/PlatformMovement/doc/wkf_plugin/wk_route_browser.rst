Route Browser - Warlock
=======================

The Route Browser displays the selected platform's route and all global routes defined in the scenario.

.. note::
   Global routes defined as part of the scenario (prior to the start of the simulation) may not be edited.

.. note::
   Only routes that have been created within Warlock are able to be deleted.

File Menu - Mystic
------------------

.. image:: images/mystic_file_menu.png

* Open... - Browse to open an :command:`event_pipe` file.
* Open Recent - Choose from a recently opened :command:`event_pipe` file.
* Save Configuration - Save the current :doc:`application settings<mystic_user_configurations>` to file.
* Load Configuration - Load a saved :doc:`application settings<mystic_user_configurations>` file.
* Import Configuration Options... - Import some features from an :doc:`application settings<mystic_user_configurations>` file.
* Recent Configurations - Choose from a recently used :doc:`application settings<mystic_user_configurations>` file.
* Clear Platform Options - Clear out all of the :doc:`platform options<mystic_platform_options>`.
* Exit - Exit the application.

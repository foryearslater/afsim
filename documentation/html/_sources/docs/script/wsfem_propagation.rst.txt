WsfEM_Propagation
-----------------

.. class:: WsfEM_Propagation inherits WsfObject

WsfEM_Propagation is the base script class for all propagation models. Currently, none of the propagation models are available in AFSIM.

This base class only provides the functionality available via :class:`WsfObject` at this time.

Mover Creator 0.2 Beta - 08 Mar 2019
------------------------------------

General
=======

This is the second beta release of Mover Creator. It is compatible with AFSIM V2.3.1 (Mission, Warlock, Results Vis, and Wizard).

* Updated Qt from 5.9.4 to 5.9.7 to fix critical vulnerabilities.

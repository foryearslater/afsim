Measure of Effectiveness Grid Data Output
-----------------------------------------

.. parsed-literal::

   output data
      file_ <file-name>
      width_ <integer>
      precision_ <integer>
      format_ ...
      justify_ ...
   end_output

Grid data output will produce a file containing the final value of the
measure of effectiveness for each grid point in the coverage grid. See the
individual measures of effectiveness for the meaning of the values measured.

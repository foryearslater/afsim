System Requirements
===================

The recommended system requirements for AFSIM are listed below. These guidelines represent an ideal environment in which to run AFSIM, but AFSIM may run on systems that do not meet these specifications.

.. csv-table:: **Recommended System Requirements**
   :widths: auto

   "Operating System", "Windows 10 or Linux RHEL/CentOS 7 (64 bit)"
   "Processor", "Quad-core @ 3.0+ GHz"
   "Memory", "8+ GB RAM"
   "Disk Space", "2+ GB"
   "Graphics Hardware", "Any GPU with OpenGL 3.0 Support"

.. note::
   The GPU requirement is to run graphical applications such as :doc:`../wizard`, :doc:`../warlock` and :doc:`../mystic`. It is not necessary to run the :doc:`core applications<../wsf_core>`.

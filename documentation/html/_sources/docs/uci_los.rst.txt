UCI_LOS
-------

.. class:: UCI_LOS

Overview
========

This class defines a sensors current line of sight (LOS).

Methods
=======

.. method:: double Azimuth()

   Returns the azimuth value with respect to the :class:`UCI_LOS_Reference` coordinate system.

.. method:: double Elevation()

   Returns the elevation value with respect to the :class:`UCI_LOS_Reference` coordinate system.

.. method:: UCI_LOS_Reference Reference()

   Returns the reference coordinate system value.

WSF_FUEL¶

fuel WSF_FUEL¶
fuel <name> WSF_FUEL
   ... Platform Part Commands ...
   ... common fuel commands ...
    consumption_rate ...
   filter Commands ...
end_fuel

Commands

Overview¶

WSF_FUEL implements a fuel object that consumes fuel at a fixed rate. See WsfFuel for script methods.

Commands¶

In addition to all of the standard fuel commands, WSF_FUEL adds the following:

consumption_rate <mass-flow-value>¶

Defines the fixed rate at which fuel will be consumed.

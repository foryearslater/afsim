Demos 2.7 - 30 Oct 2020¶

Visit the AFSIM 2.7 Release Page on DI2E (Access Controlled) for more information and resources related to this release.

Release Dates¶

  • Demos 2.7.3 - 24 Sep 2021

  • Demos 2.7.2 - 12 Mar 2021

  • Demos 2.7.1 - 19 Nov 2020

  • Demos 2.7.0 - 30 Oct 2020

General¶

  • Object types and instances (including platforms, sensors, etc.) were renamed throughout the demos to be more generic, to avoid misrepresenting actual systems. (Issue #130)

  • Added alternate_locations demo, which demonstrates the capabilities of the AFSIM alternate_locations command block.

  • Added the run_tree scenario to the behavior_tree demos. (Issue #125)

  • Added the air_to_air demo, which provides several scenarios that demonstrate air-to-air combat using the WSF_SA_PROCESSOR and Advanced Behavior Trees (ABTs).

  • Updated the p6dof demos as a result of the changed defaults for aero_data legacy flags use_reduced_frequency and use_legacy_derivatives. (Issue #124)

Corrections¶

Enhancements¶

  • Removed uses of WsfSpaceMover.Boost and WsfSpaceMover.AddDeltaV. (Issue #126)

  • Changed the output in noise_cloud to output a geotiff file in place of the PD file.

  • Updated the proximity operations demo in space_operations to use the updated target command. (Issue #122)

  • Added a set of coverage_demos. (Issue #128)

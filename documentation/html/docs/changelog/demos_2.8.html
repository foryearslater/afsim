Demos 2.8 - 11 Mar 2021¶

Visit the AFSIM 2.8 Release Page on DI2E (Access Controlled) for more information and resources related to this release.

Release Dates¶

  • Demos 2.8.2 - 22 Oct 2021

  • Demos 2.8.1 - 29 Sep 2021

  • Demos 2.8.0 - 11 Mar 2021

General¶

  • Additional demos have been added to compare the new WsfPointMassSixDOF_Mover to the WsfBrawlerMover. (AFSIM-1002)

  • Demos have been added for the new WsfSixDOF_Mover classes. (AFSIM-662)

  • Added a 2v2 air_to_air combat demo variant which uses the new WSF_POINT_MASS_SIX_DOF_MOVER.

  • Added the state_machine scenario to the behavior_tree demos. (AFSIM-667)

  • Added the subtrees scenario to the behavior_tree demos. (AFSIM-666)

Corrections¶

  • (2.8.1) Modified aerobraking from satellite_demos in response to a correction made to WSF_INTEGRATING_SPACE_MOVER, and to make use of periapsis_altitude, and apoapsis_altitude. (AFSIM-1720)

Enhancements¶

  • Added multi-player DIS demo in wargame_demo. (AFSIM-434)

  • (2.8.1) Updated gun_engagement to more robustly handle a user-provided bullet name. (AFSIM-197)

  • Added aerobraking demo in satellite_demos. (AFSIM-535)

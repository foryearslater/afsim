Mover Creator 2.7 - 30 Oct 2020¶

Visit the AFSIM 2.7 Release Page on DI2E (Access Controlled) for more information and resources related to this release.

Release Dates¶

  • Mover Creator 2.7.3 - 24 Sep 2021

  • Mover Creator 2.7.2 - 12 Mar 2021

  • Mover Creator 2.7.1 - 19 Nov 2020

  • Mover Creator 2.7.0 - 30 Oct 2020

General¶

Known Issues¶

Corrections¶

  • Fixed issue where the performance tests would calculate zero thrust for unattainable aerodynamic conditions. (Issue #190)

  • Fixed issue where the autopilot support dialog would indicate that it was complete before the process was finished. (Issue #191)

  • Fixed issue in the level flight range performance test where the test weapon was launched with a launch speed of zero, rather than the intended speed of Mach 0.85. (Issue #192)

  • Corrected a units conversion issue in the endurance and range performance tests. (Issue #192)

  • Fixed issue where Mover Creator would unexpectedly exit if the user attempted to create a new vehicle or engine from a malformed template file. (Issue #194)

  • Fixed issue where running Mover Creator on Linux would sometimes crash the application when performing a variety of actions. (Issue #199)

  • Fixed issue where performance tests would not run on Linux. (Issue #201)

  • Fixed erroneous calculation of surface area for Bodies. (Issue #203)

Enhancements¶

  • Performance tests can now calculate drag at a Mach number of zero. (Issue #190)

  • Added support for the WSF_GUIDED_MOVER. See the updated User’s Guide for more information. (Issue #188)

  • The Pilots/Controls tab now appears after the Performance tab to improve overall application flow. (Issue #188)

  • The performance test page now allows the user to select individual series to display on the plot. (Issue #196)

  • The test description section on the performance page now contains timestamp information, indicating when the test was last executed. (Issue #197)

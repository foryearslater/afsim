AFSIM 2.6 - 08 May 2020¶

Visit the AFSIM 2.6 Release Page on DI2E (Access Controlled) for more information and resources related to this release.

Release Dates¶

  • AFSIM 2.6.3 - 19 Feb 2021

  • AFSIM 2.6.2 - 13 Nov 2020

  • AFSIM 2.6.1 - 05 Aug 2020

  • AFSIM 2.6.0 - 08 May 2020

Security Updates¶

  • (2.6.3) Corrected some memory leaks within XIO and the packetio library exposed through the script method WsfXIO.Subscribe. (AFSIM-951)

  • (2.6.2) Resolved potential memory leaks due to thread safety issues in the terrain model. (AFSIM-312)

  • (2.6.3) Addressed multiple security vulnerabilities in SDL by updating from version 2.0.7 to 2.0.14. Affects Warlock only. (AFSIM-962)

    • CVE-2019-7572, CVE-2019-7573, CVE-2019-7574, CVE-2019-7575, CVE-2019-7576, CVE-2019-7577, CVE-2019-7578, CVE-2019-7635, CVE-2019-7636, CVE-2019-7637, CVE-2019-7638, CVE-2019-13616, CVE-2019-13626, CVE-2019-14906, CVE-2020-14409, CVE-2020-14410

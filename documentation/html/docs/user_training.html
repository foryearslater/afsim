Behavior and Agent Modeling¶

Behavior and Agent Modeling (BAM) training is an advanced training package with the following modules:

  • 21-BAM Overview

  • 22-Perception and Quantum Tasker Processors

  • 23-Behavior Trees and Pursue Behavior

  • 24-Route Finder

  • 25-Go Home and Escort Behaviors

  • 26-Threat Processor and Evade Behavior

  • 27-Clustering, C2 and Decomposing Tasks

  • 28-Integrated Air and Ground Defenses

Mystic¶

The Mystic training package provides an overview of the Mystic application with the following modules:

  • 12-Mystic Overview

  • 14-Mystic and Warlock Common Features

  • 15-Mystic Features

Warlock¶

The Warlock training package provides an overview of the Warlock application with the following modules:

  • 13-Warlock Overview

  • 14-Warlock and Mystic Common Features

  • 16-Warlock Features

Core¶

Core training is an introductory training package with the following modules:

  • 1-Overview

  • 2-Applications and Tools

  • 3-Platform Types and Platforms

  • 4-Movers and Routes

  • 5-Weapons and Execute

  • 6-Signatures and Part Manager

  • 7-Sensors and Tracks

  • 8-Scripting Language and Script Processors

  • 9-WsfDraw and Script Observer

  • 10-Task Processor State Machines

  • 11-Command Chain and Communications

Space¶

The Space training package provides an overview of the AFSIM space capabilities with the following modules:

  • 1-Constellations and Coverage

  • 2-Satellite Communications

  • 3-Space Situational Awareness (SSA)

  • 4-Rendezvous and Proximity Operations

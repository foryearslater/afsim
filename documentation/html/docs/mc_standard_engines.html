Mover Creator Standard Engine Templates¶

Mover Creator provides a set of predefined standard engine templates that can be used for any Aircraft or Weapon.

Page Navigation¶

The engine templates described here are divided into two categories: Nominal and Refined.

“Nominal” templates are those that are not being used in any particular vehicle template. Typically, when a user creates a new engine, they should be deriving their engine from one of these “nominal” engine templates.

“Refined” templates are those that have been created specifically for an existing vehicle template.

  • Nominal Engine Templates

    • Jet

    • Ramjet

    • Liquid Rocket

    • Solid Rocket

  • Refined Engine Templates

    • Jet

    • Ramjet

    • Liquid Rocket

    • Solid Rocket

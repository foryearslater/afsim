WSF_MULTIRESOLUTION_FUEL¶

multiresolution WSF_MULTIRESOLUTION_FUEL¶
multiresolution_fuel WSF_MULTIRESOLUTION_FUEL
   ... multiresolution_fuel ...
end_multiresolution_fuel

Overview¶

WSF_MULTIRESOLUTION_FUEL is a container for holding one or more fuel models on a platform and deferring the choice of which fuel to use on the platform until run-time. It is a multiresolution_fuel type and supports all inputs defined in that command.

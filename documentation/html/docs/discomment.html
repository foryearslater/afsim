DisComment¶

DisComment inherits DisPdu
DisComment.ReceivingEntity
DisComment.OriginatingEntity
DisComment.String

Overview¶

DisComment is an implementation of the DIS comment PDU. Comment PDUs are used to pass information between platforms in the simulation environment.

Methods¶

DisEntityId ReceivingEntity()¶

Returns the ID of the platform receiving the comment.

DisEntityId OriginatingEntity()¶

Returns the ID of the platform sending the comment.

string String()¶

Returns the contents of the comment.

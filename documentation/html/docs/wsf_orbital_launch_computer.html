WSF_ORBITAL_LAUNCH_COMPUTER¶

launch_computer WSF_ORBITAL_LAUNCH_COMPUTER¶
launch_computer <name> WSF_ORBITAL_LAUNCH_COMPUTER
   ... launch_computer Commands ...
   ... WSF_LAUNCH_COMPUTER Commands ...
   ... WSF_ORBITAL_LAUNCH_COMPUTER Commands ...
end_launch_computer

Overview¶

WSF_ORBITAL_LAUNCH_COMPUTER implements a launch computer for an orbital launch vehicle.

Tables for this launch computer should be generated with the weapon_tools utility. See ORBITAL_LAUNCH_COMPUTER_GENERATOR.

Commands¶

leo_data <leo_data_file_name>¶

Specify the file name to use when computing a low-Earth orbit (LEO) launch solution. This file is created using the ORBITAL_LAUNCH_COMPUTER_GENERATOR command of the weapon_tools utility.

Default: None, must be specified.

WsfOrbitalManeuver¶

WsfOrbitalManeuver inherits WsfOrbitalEvent

WsfOrbitalManeuvers are specific orbital events, utilized by a WsfSpaceMover to change the orbit of a platform, either directly, using WsfSpaceMover.ExecuteEvent, or as part of a mission sequence , using WsfSpaceMover.ExecuteMissionSequence. Each script orbital maneuver type is meant to be configured with a WsfOrbitalEventCondition object, passed in the object’s Construct method.

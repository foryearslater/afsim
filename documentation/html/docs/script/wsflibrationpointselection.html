WsfLibrationPointSelection¶

WsfLibrationPointSelection

Overview¶

The WsfLibrationPointSelection class provides a means to select for which libration point information is being queried in the WsfLibrationPoint.

Static Methods¶

WsfLibrationPointSelection L1()¶

Return a WsfLibrationPointSelection instance representing the L1 point.

WsfLibrationPointSelection L2()¶

Return a WsfLibrationPointSelection instance representing the L2 point.

WsfLibrationPointSelection L3()¶

Return a WsfLibrationPointSelection instance representing the L3 point.

WsfLibrationPointSelection L4()¶

Return a WsfLibrationPointSelection instance representing the L4 point.

WsfLibrationPointSelection L5()¶

Return a WsfLibrationPointSelection instance representing the L5 point.

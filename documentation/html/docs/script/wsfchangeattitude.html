WsfChangeAttitude¶

WsfChangeAttitude inherits WsfOrbitalEvent

Input type: change_attitude

WsfChangeAttitude is used to change the platform’s orientation to one of the supported orientation types.

static WsfChangeAttitude Construct(WsfOrbitalEventCondition aCondition, WsfSpaceOrientation aOrientation)¶

Static method to create a WsfChangeAttitude event using a WsfSpaceOrientation parameter.

Note

If an entity or geo-point orientation type is specified (WsfSpaceOrientation.ENTITY_WITH_SOLAR_CONSTRAINT, WsfSpaceOrientation.ENTITY_WITH_NADIR_CONSTRAINT, WsfSpaceOrientation.ENTITY_WITH_ORBIT_PLANE_CONSTRAINT, or WsfSpaceOrientation.POINT_WITH_ORBIT_PLANE_CONSTRAINT) and the referenced WsfPlatform or WsfGeoPoint is invalid, the event will not execute.

WsfEM_Antenna¶

WsfEM_Antenna

WsfEM_Antenna is the geometric representation of an antenna, providing the physical location and orientation.

Methods¶

WsfArticulatedPart ArticulatedPart()¶

Returns the articulated part associated with the antenna.

double Height()¶

Returns the height (m) of the antenna with respect to the host WsfArticulatedPart.

void SetHeight(double aHeight)¶

Sets the height (m) of the antenna with respect to the host WsfArticulatedPart.

double Pitch()¶

Returns the antenna pitch angle above/below the scan plane of the host WsfArticulatedPart. (degrees)

void SetPitch(double aPitch)¶

Sets the antenna pitch angle above/below the scan plane of the host WsfArticulatedPart. (degrees)

WsfFieldOfView FieldOfView()¶

Return the field of view of the antenna.

void SetFieldOfView(WsfFieldOfView aFieldOfView)¶

Apply a field of view to the antenna.

Array<double> AzimuthFieldOfView()¶

Returns the angular (degrees) azimuth field of view limits from a rectangular field of view. The first value is the minimum azimuth field of view limit, while the second value is the maximum azimuth field of view limit.

void SetAzimuthFieldOfView(double aMinAz, double aMaxAz)¶

Sets the angular (degrees) azimuth field of view limits to a rectangular field of view. The first parameter is the minimum azimuth field of view limit, while the second value is the maximum azimuth field of view limit.

Array<double> ElevationFieldOfView()¶

Returns the angular (degrees) elevation field of view limits from a rectangular field of view. The first value is the minimum elevation field of view limit, while the second value is the maximum elevation field of view limit.

Sets the angular (degrees) elevation field of view limits to a rectangular field of view. The first parameter is the minimum elevation field of view limit, while the second value is the maximum elevation field of view limit.

string ScanMode()¶

Returns the scan mode for this antenna. The following modes are available:

  • azimuth

  • elevation

  • both (azimuth and elevation scanning)

bool SetScanMode(string aMode)¶

  • azimuth

  • elevation

  • both (azimuth and elevation scanning)

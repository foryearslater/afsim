WsfOrbitalBlendedCost¶

WsfOrbitalBlendedCost inherits WsfOrbitalOptimizationCost

WsfOrbitalBlendedCost represents an orbital targeting optimization cost function that is a polynomial that blends together the elapsed time, , and the total change in speed, . The value of the cost function is set by three constants, , , and as follows:

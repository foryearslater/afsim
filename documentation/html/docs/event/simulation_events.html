Simulation Events

These are the event_output events related to the WsfSimulation.

SIMULATION_COMPLETE¶

event_output Signature:

<time> <event> <year> <month> <day> <hour> <minute> <second>

Signature Elements:

Fields

Description

<time>

The current simulation time

<event>

The event name

<year>

Simulation year for the current simulation time

<month>

Simulation month for the current simulation time

<day>

Simulation second for the current simulation time

<hour>

Simulation hour for the current simulation time

<minute>

Simulation minute for the current simulation time

<second>

Simulation second for the current simulation time

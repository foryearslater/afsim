Help - Mystic¶

The Help menu has links to the About dialog and the Documentation dialog.

The About dialog has information about the version, third-party licenses, artwork attributions, and advanced information about the system.

The Documentation dialog contains links to the important parts of the AFSIM documentation.

Command Line¶

Mystic will display Command Line Usage when the command line argument of -?, -h, or -help is supplied when executing. This will show a dialog that displays proper command line usage of Mystic.

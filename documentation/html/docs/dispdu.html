Overview¶

DisPdu is an implementation of a generic DIS PDU. PDUs are used to communicate information between simulations on a DIS network.

Methods¶

int ProtocolVersion()¶

Returns the DIS Protocol version.

int Exercise()¶

Returns the DIS exercise ID.

int PduType()¶

Returns the type of PDU.

1. DisEntityState
2. DisFire
3. DisDetonation
22. DisComment
23. DisEmission
25. DisTransmitter
27. DisReceiver
int ProtocolFamily()¶

Returns the DIS Protocol family.

int TimeStamp()¶

Returns the timestamp for the PDU.

int BaseLength()¶

Returns the length of the generic DIS PDU, excluding type-specific information.

int Length()¶

Returns the length of the entire DIS PDU, including type-specific information.
